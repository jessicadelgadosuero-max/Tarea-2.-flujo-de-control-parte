# Tarea 2. flujo de control parte

1. Descripción:

Este programa fue desarrollado en C++ utilizando Visual Studio.

Su función es leer el sueldo mensual de un empleado, calcular el Impuesto Sobre la Renta (ISR) correspondiente al año 2026 según la escala salarial publicada por la Dirección General de Impuestos Internos (DGII) de la República Dominicana.

Si el empleado no debe pagar ISR, el programa muestra un descuento de RD$0.00 (N/A).
